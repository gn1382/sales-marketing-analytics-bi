# Sales & Digital Marketing Analytics (BI Project)

This is an end-to-end analytics project designed to practice and demonstrate
Business Intelligence and Data Analytics skills.

## Project Goals
- Analyze sales performance
- Identify top products
- Measure digital marketing campaign effectiveness
- Track KPIs and business metrics

## Tech Stack
- SQL Server (T-SQL)
- Python (pandas, basic ML later)
- Power BI
- Excel
- Git & GitHub

## Project Structure
- ERD Design (Day 1)
- Database implementation (SQL Server)
- Data cleaning & transformation
- KPI definition
- Dashboard & reporting (Power BI)

## Status
- Day 1: ERD design completed
- Day 2: Relational database schema implemented in SQL Server using T-SQL
